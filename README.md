# Digital Consciousness — Particle Face WebGL

An interactive WebGL / Three.js experience featuring a human face generated by over 50,000 dynamic particles. This project explores "Neural Core" rendering and fluid interaction via custom shaders (GLSL).

## 🌟 Key Features

- **Procedural Point Cloud Rendering**: 50,000+ particles merged to recreate the illusion of a digital consciousness with a holographic/scanner effect (Additive Blending).
- **Shader Physics (GLSL)**: Assembly animation, gravity/falling effect on the shoulders, and local lighting are calculated directly on the GPU for zero-latency 60+ FPS performance.
- **Interactive Gaze Tracking**: Clamped biometric mouse tracking. The physical model of the neck and head rotates organically while keeping the body/chest stationary.
- **Modern UI Interface (Glassmorphism HUD)**: Overlay of visual elements typical of "Tech" dashboards (Interactive cursor, latency tracking, JetBrains Mono typography).

## 🚀 How to Run Locally

This project uses **Vite** as a build tool for rapid development.

### 1. Prerequisites
- [Node.js](https://nodejs.org/en/) (version 16+ recommended)

### 2. Installation
Clone the repository and install the npm dependencies:

```bash
# Install modules (Three.js, Vite, etc.)
npm install
```

### 3. Start Development Server
Once installed, start the local server:

```bash
npm run dev
```

The terminal will show an address (usually `http://localhost:5173`). Open it in your browser to experience the real-time render!

### 4. Build for Production
To generate an optimized static version, ready for deployment (Vercel, Netlify, GitHub Pages...):

```bash
npm run build
```
The minified files will be located in the `dist/` directory.
